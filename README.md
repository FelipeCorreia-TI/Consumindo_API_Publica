# 📋 Lista de Usuários - Consumo de API Pública

Este projeto é uma aplicação simples em HTML, CSS e JavaScript que consome uma API pública para exibir uma lista de usuários e seus respectivos detalhes.

## 🚀 Funcionalidades

- 🔍 Busca de usuários a partir de uma API pública
- 📄 Exibição dinâmica da lista de usuários
- 👤 Visualização de detalhes ao clicar em um usuário
- 🌐 Link direto para o website do usuário
- ⚠️ Tratamento básico de erros na requisição

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- API pública: https://jsonplaceholder.typicode.com/users
